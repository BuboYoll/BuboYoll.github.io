# 冯菁源的个人网站

这是我的个人网站源代码，使用GitHub Pages和Jekyll构建。

## 网站特点

- 极简风格设计
- 响应式布局，适配各种设备
- 包含首页、关于我和博客页面
- 动画和交互效果

## 技术栈

- GitHub Pages
- Jekyll
- HTML/CSS
- JavaScript

## 本地开发

如果你想在本地运行这个网站，请按照以下步骤操作：

1. 克隆仓库
   ```
   git clone https://github.com/BuboYoll/BuboYoll.github.io.git
   cd BuboYoll.github.io
   ```

2. 安装Jekyll和依赖
   ```
   gem install bundler jekyll
   bundle install
   ```

3. 启动本地服务器
   ```
   bundle exec jekyll serve
   ```

4. 在浏览器中访问 `http://localhost:4000`

## 部署

这个网站会自动部署到GitHub Pages。只需将更改推送到主分支即可。
